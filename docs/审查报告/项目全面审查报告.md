## 一、问题清单（按严重程度排序）

### 🔴 严重问题 (P0) - 13 个

#### 1. 资源泄漏风险 ⚠️⚠️⚠️
**严重程度**: 🔴🔴🔴 最高  
**影响范围**: 所有插件  
**问题**: 所有插件都缺少完整的 `destroy` 实现，响应式引用和 watch 监听器没有清理

**证据**:
```typescript
// src/MapLibre/plugins/map-feature-snap/useMapFeatureSnapPlugin.ts:54-56
destroy: () => {
  pluginController.binding.value?.destroy?.();
  // ❌ 缺少 watch 清理、响应式引用清理
}
```

**影响**:
- 插件卸载后，Vue 响应式系统仍持有引用
- 频繁切换插件配置时会导致内存泄漏
- `watch` 回调可能在插件销毁后仍被触发

**修复建议**:
```typescript
// 在 useMapFeatureSnapController 中返回清理函数
export function useMapFeatureSnapController(options) {
  const stopWatch = watch(/* ... */);
  
  return {
    destroy: () => {
      stopWatch?.();
      bindingRef.value?.destroy?.();
      bindingRef.value = null;
    }
  };
}
```

**预计修复时间**: 4 小时  
**优先级**: P0 - 立即修复

#### 2. 插件错误隔离不足 ⚠️⚠️⚠️
**严重程度**: 🔴🔴🔴 最高  
**影响范围**: 整个地图组件  
**问题**: 单个插件初始化或运行时错误会导致整个地图组件失败

**证据**:
```typescript
// src/MapLibre/core/useMapPluginHost.ts:481-506
function createPluginRecord(descriptor: AnyMapPluginDescriptor): MapPluginRecord {
  try {
    const instance = descriptor.plugin.createInstance(context);
    return pluginRecord;
  } catch (error) {
    console.error(`[MapPluginHost] 插件 '${descriptor.id}' 初始化失败`, error);
    throw error; // ❌ 错误会向上传播，导致整个地图初始化失败
  }
}
```

**影响**:
- 单个插件初始化失败会导致整个地图组件挂载失败
- 插件运行时错误没有被捕获
- 用户体验差：一个非核心插件出错会让整个地图不可用

**修复建议**:
```typescript
function createPluginRecord(descriptor: AnyMapPluginDescriptor): MapPluginRecord | null {
  try {
    const instance = descriptor.plugin.createInstance(context);
    
    // 包装插件方法，捕获运行时错误
    const safeInstance: MapPluginInstance = {
      getRenderItems: () => {
        try {
          return instance.getRenderItems?.() || [];
        } catch (error) {
          console.error(`[MapPluginHost] 插件 '${descriptor.id}' getRenderItems 失败`, error);
          return [];
        }
      },
      // ... 其他方法同样包装
    };
    
    return { descriptorRef, instance: safeInstance };
  } catch (error) {
    console.error(`[MapPluginHost] 插件 '${descriptor.id}' 初始化失败，已跳过`, error);
    return null; // 返回 null 而不是抛出异常
  }
}
```

**预计修复时间**: 6 小时  
**优先级**: P0 - 立即修复

#### 4. useBusinessMap 缺少地图实例空值保护 ⚠️⚠️
**严重程度**: 🔴🔴 高  
**影响范围**: 运行时稳定性  
**问题**: 业务层在地图未就绪时调用门面方法会导致运行时崩溃

**证据**:
```typescript
// src/MapLibre/facades/useBusinessMap.ts:170-194
export function useBusinessMap(options: UseBusinessMapOptions): UseBusinessMapResult {
  const { mapRef, sourceRegistry } = options;
  
  // ❌ 直接传递 mapRef，未做空值校验
  const selection = useMapSelection(mapRef);
  const feature = createBusinessMapFeatureGroup(options);
}
```

**修复建议**:
```typescript
export function useBusinessMap(options: UseBusinessMapOptions): UseBusinessMapResult {
  const { mapRef, sourceRegistry } = options;
  
  // 添加空值保护
  const safeMapRef = computed(() => {
    const map = toValue(mapRef);
    if (!map) {
      console.warn('[useBusinessMap] 地图实例尚未就绪');
    }
    return map;
  });
  
  const selection = useMapSelection(safeMapRef);
  // ...
}
```

**预计修复时间**: 2 小时  
**优先级**: P0 - 立即修复
