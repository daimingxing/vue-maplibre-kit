# 自定义控件

适合谁读：需要把业务按钮、图例或简单工具放进地图控件区域的开发者。

先读哪篇：[03-mapControls](./03-mapControls.md)。

对应示例：当前没有独立自定义控件示例；可结合 [NGGI01](../../../examples/views/NG/GI/NGGI01.vue) 的控件配置理解。

## 推荐方式

从业务入口使用公开控件组件：

```vue
<template>
  <MapLibreInit :map-options="mapOptions" :controls="controls">
    <MglCustomControl position="top-right">
      <button type="button" @click="zoomToWorkArea">定位</button>
    </MglCustomControl>
  </MapLibreInit>
</template>

<script setup lang="ts">
import { MapLibreInit, MglCustomControl } from "vue-maplibre-kit/business";

/**
 * 定位到业务工作区。
 */
function zoomToWorkArea(): void {
  // 这里接入业务自己的定位逻辑。
}
</script>
```

## 使用建议

- 简单按钮、图例开关、定位动作适合放进自定义控件。
- 复杂表单、属性面板、结果列表更适合放在地图外侧业务面板。
- 自定义控件内调用命令式地图能力时，仍建议通过 `useBusinessMap` 取得稳定入口。
