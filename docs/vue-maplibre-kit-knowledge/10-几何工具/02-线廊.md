# 线廊

线廊能力由 `MapLineCorridorTool` 提供。它把一条中心线按半宽生成一个 Polygon 面，适合道路红线、管廊范围、缓冲展示等业务场景。

## 常用方法

| 方法 | 用途 |
| --- | --- |
| `createRegionFeature(lineFeature, widthMeters, options)` | 根据中心线创建单个线廊面 |
| `replaceRegionFeatures(features, lineFeature, widthMeters, options)` | 在要素集合中替换同一条线已生成的旧线廊 |
| `isGeneratedRegionFeature(feature, lineId, generatedKind)` | 判断某个面是否为指定线生成的线廊 |

## 生成单个线廊

```ts
import { MapLineCorridorTool } from 'vue-maplibre-kit/geometry';

/**
 * 根据中心线生成管廊面。
 * @param lineFeature 中心线要素
 * @param widthMeters 中心线到边界的距离，单位米
 * @returns 线廊面要素；几何无效时返回 null
 */
export function buildCorridor(lineFeature, widthMeters: number) {
  return MapLineCorridorTool.createRegionFeature(lineFeature, widthMeters, {
    regionType: '管廊区域',
  });
}
```

## 替换旧线廊

`replaceRegionFeatures()` 会先删除同一条线、同一 `generatedKind` 已生成的区域，再追加新区域。这样可以避免用户多次调整宽度后出现多块重复面。

```ts
import { MapLineCorridorTool } from 'vue-maplibre-kit/geometry';

/**
 * 用最新宽度替换指定中心线对应的线廊。
 * @param features 当前 source 内的完整要素列表
 * @param lineFeature 中心线要素
 * @param widthMeters 中心线到边界的距离，单位米
 * @returns 替换后的完整要素列表；无法生成时返回 null
 */
export function replaceCorridor(features, lineFeature, widthMeters: number) {
  return MapLineCorridorTool.replaceRegionFeatures(features, lineFeature, widthMeters);
}
```

## 生成结果

生成的面要素会带上这些业务字段：

- `generatedKind`：默认值为 `line-corridor`。
- `generatedFromLineId`：来源线的业务 ID。
- `widthMeters`：本次传入的半宽。
- `managedPreviewOrigin*`：如果来源线带有托管预览来源属性，会继续传递给生成面。

## 边界

- `widthMeters` 必须大于 `0`，否则业务语义无效。
- 中心线至少需要两个有效坐标点。
- 线廊工具只返回 GeoJSON，不负责写回 source。
- 业务页需要写回正式 source 时，优先调用 `businessMap.feature.replaceLineCorridor()` 或 `businessMap.feature.replaceSelectedLineCorridor()`。
