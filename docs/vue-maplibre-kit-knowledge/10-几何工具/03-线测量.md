# 线测量

线测量能力由 `MapLineMeasureTool` 提供。它面向普通业务线，不依赖 TerraDraw 测量控件。

## 常用方法

| 方法 | 用途 |
| --- | --- |
| `getDistanceInMeters(start, end)` | 计算两个坐标点之间的球面距离 |
| `getFeatureLengthInMeters(feature)` | 计算线要素总长度 |
| `getCoordinatesLengthInMeters(coordinates)` | 计算折线坐标数组总长度 |
| `measureFeatureBetweenCoordinates(feature, start, end)` | 计算线要素内两点之间的沿线距离 |
| `measureBetweenCoordinates(coordinates, start, end)` | 计算折线内两点之间的沿线距离 |

## 整线长度

```ts
import { MapLineMeasureTool } from 'vue-maplibre-kit/geometry';

/**
 * 读取线要素总长度。
 * @param feature 当前业务要素
 * @returns 长度文本
 */
export function formatLineLength(feature) {
  const length = MapLineMeasureTool.getFeatureLengthInMeters(feature);
  if (length === null) {
    return '不是线要素';
  }

  return `${length.toFixed(2)} m`;
}
```

## 线内区间长度

`measureFeatureBetweenCoordinates()` 会把输入点投影到最近线段，再返回两点沿线之间的距离。

```ts
import { MapLineMeasureTool } from 'vue-maplibre-kit/geometry';

/**
 * 测量两次点击点在线内的里程差。
 * @param lineFeature 当前线要素
 * @param start 起点经纬度坐标
 * @param end 终点经纬度坐标
 * @returns 沿线距离；无法测量时返回 null
 */
export function measureLinePart(lineFeature, start, end) {
  return MapLineMeasureTool.measureFeatureBetweenCoordinates(lineFeature, start, end);
}
```

返回结果包含：

- `start.segmentIndex`、`end.segmentIndex`：起终点投影到的线段索引。
- `start.distanceFromStartMeters`、`end.distanceFromStartMeters`：从整线起点累计到投影点的里程。
- `lengthMeters`：两点沿线之间的绝对距离。

## 边界

- 非 `LineString` 要素返回 `null`。
- 连续重复坐标会被过滤，避免零长度线段影响定位。
- 距离单位固定为米。
- 该工具计算的是沿线二维距离，不等同于 TerraDraw 测量控件的地形高程距离。
