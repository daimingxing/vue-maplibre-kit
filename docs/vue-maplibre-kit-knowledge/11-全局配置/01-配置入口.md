# 配置入口

公开入口：

`	s
import {
  defineMapGlobalConfig,
  setMapGlobalConfig,
  getMapGlobalConfig,
  resetMapGlobalConfig,
} from 'vue-maplibre-kit/config';
`

defineMapGlobalConfig() 只提供类型辅助，不产生运行时副作用；setMapGlobalConfig() 会注册当前应用的只读全局配置快照。

## 真实项目中的用法

### 2.1 推荐目录

```txt
src/
  main.ts
  map-global-config.ts
```

### 2.2 推荐写法

`src/map-global-config.ts`

```ts
import { defineMapGlobalConfig, setMapGlobalConfig } from 'vue-maplibre-kit/config';

/**
 * 应用级地图全局默认配置。
 * 推荐把全局默认值集中写到这个文件里，再在 main.ts 启动阶段注册一次。
 */
export const mapGlobalConfig = defineMapGlobalConfig({
  mapOptions: {
    mapStyle: 'https://demotiles.maplibre.org/style.json',
    center: [113.943, 22.548],
    zoom: 8,
  },
  mapControls: {
    MglNavigationControl: {
      isUse: true,
      position: 'top-left',
      showCompass: true,
      showZoom: true,
    },
  },
  plugins: {
    snap: {
      defaultTolerancePx: 12,
      preview: {
        pointColor: '#1677ff',
        lineColor: '#1677ff',
      },
    },
    lineDraft: {
      styleOverrides: {
        line: {
          paint: {
            'line-opacity': 0.8,
          },
        },
      },
    },
    intersection: {
      visible: true,
      scope: 'all',
      includeEndpoint: false,
      materializedStateStyles: {
        selected: {
          color: '#1677ff',
        },
      },
    },
    polygonEdge: {
      style: {
        normal: {
          color: '#409eff',
          width: 3,
        },
      },
    },
    multiSelect: {
      position: 'top-right',
      deactivateBehavior: 'retain',
      closeOnEscape: true,
    },
  },
  styles: {
    line: {
      paint: {
        'line-color': '#1677ff',
      },
    },
  },
});

/**
 * 注册地图全局默认配置。
 */
export function applyMapGlobalConfig(): void {
  setMapGlobalConfig(mapGlobalConfig);
}
```

`src/main.ts`

```ts
import { createApp } from 'vue';
import App from './App.vue';
import { applyMapGlobalConfig } from './map-global-config';

applyMapGlobalConfig();

createApp(App).mount('#app');
```

---

## API 说明

| API | 作用 | 运行时副作用 |
| --- | --- | --- |
| defineMapGlobalConfig(config) | 提供类型辅助并原样返回配置对象 | 无 |
| setMapGlobalConfig(config) | 注册当前应用的全局配置快照 | 会深拷贝并冻结快照 |
| getMapGlobalConfig() | 读取当前全局配置快照 | 无 |
| esetMapGlobalConfig() | 清空当前全局配置 | 会回到空配置 |

## 冻结和响应式对象

setMapGlobalConfig() 会深拷贝并冻结传入对象。源码还会检查配置中是否包含 Vue 响应式对象或 ef，如果检测到，会输出提示：

`	xt
[setMapGlobalConfig] 检测到 Vue 响应式配置值，建议传入普通对象快照
`

全局配置适合放稳定默认值，不适合直接保存页面响应式状态。
