# 样式工厂

适合谁读：需要快速配置点线面样式，或覆盖 MapLibre paint/layout 字段的开发者。

先读哪篇：[样式与状态索引](./index.md)。

对应示例：[NGGI02](../../../examples/views/NG/GI/NGGI02.vue)、[NGGI03](../../../examples/views/NG/GI/NGGI03.vue)。

## 简单样式

```ts
import {
  createSimpleCircleStyle,
  createSimpleFillStyle,
  createSimpleLineStyle,
} from "vue-maplibre-kit/business";

const pointStyle = createSimpleCircleStyle({
  color: "#f97316",
  radius: 7,
  opacity: 0.9,
  strokeColor: "#ffffff",
  strokeWidth: 2,
});

const lineStyle = createSimpleLineStyle({ color: "#0f766e", width: 4, opacity: 0.9 });
const fillStyle = createSimpleFillStyle({ color: "#79b8ff", opacity: 0.35 });
```

## 完整样式

```ts
import {
  createCircleLayerStyle,
  createFillLayerStyle,
  createLineLayerStyle,
  createRasterLayerStyle,
  createSymbolLayerStyle,
} from "vue-maplibre-kit/business";
```

这些工厂接收 `layout` 和 `paint` 的局部覆盖，并合并默认样式。

```ts
const lineStyle = createLineLayerStyle({
  layout: { "line-cap": "round", "line-join": "round" },
  paint: { "line-color": "#2563eb", "line-width": 5 },
});
```

## 选择建议

- 点线面快速业务图层：优先用 `createSimple*Style`。
- 需要完整 MapLibre 字段：使用 `create*LayerStyle`。
- 需要按 hover、selected 或业务字段变化：样式字段可以传表达式。
