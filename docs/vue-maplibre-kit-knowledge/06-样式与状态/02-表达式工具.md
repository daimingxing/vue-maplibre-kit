# 表达式工具

适合谁读：需要按业务属性或 feature-state 动态改变样式的开发者。

先读哪篇：[01-样式工厂](./01-样式工厂.md)。

对应示例：[NGGI03](../../../examples/views/NG/GI/NGGI03.vue)。

## 读取 properties

```ts
import {
  getFeatureBoolean,
  getFeatureColor,
  getFeatureNumber,
  getFeatureProperty,
  getFeatureString,
} from "vue-maplibre-kit/business";

const rawColor = getFeatureProperty("color");
const color = getFeatureColor("color", "#79b8ff");
const width = getFeatureNumber("width", 3);
const label = getFeatureString("label", "");
const disabled = getFeatureBoolean("disabled", false);
```

`getFeatureProperty()` 是低阶工具，只生成原生 `get` 表达式。`getFeatureColor()`、`getFeatureNumber()`、`getFeatureString()`、`getFeatureBoolean()` 是业务推荐写法，会同时处理默认值和类型转换。

## feature-state 表达式

```ts
import { createFeatureStateExpression, getFeatureColor } from "vue-maplibre-kit/business";

const color = createFeatureStateExpression({
  default: getFeatureColor("color", "#f97316"),
  hover: getFeatureColor("hoverColor", "#facc15"),
  selected: "#2563eb",
  states: {
    active: "#ef4444",
  },
  order: ["active", "selected", "hover"],
});
```

`order` 越靠前优先级越高。示例中 `active` 会覆盖 `selected` 和 `hover`。

表达式工具可以嵌套。上面的默认颜色读取 `properties.color`，hover 颜色读取 `properties.hoverColor`，selected 仍使用固定颜色。

## 属性等值

```ts
import { whenFeaturePropertyEquals } from "vue-maplibre-kit/business";

const color = whenFeaturePropertyEquals("status", "warning", "#ef4444", "#22c55e");
```

命中值和默认值也可以继续嵌套属性取值表达式：

```ts
const color = whenFeaturePropertyEquals(
  "status",
  "warning",
  getFeatureColor("warningColor", "#ef4444"),
  getFeatureColor("color", "#22c55e")
);
```

## 属性多值命中

```ts
import { whenFeaturePropertyIn } from "vue-maplibre-kit/business";

const width = whenFeaturePropertyIn("status", ["warning", "checking"], 6, 3);
```

```ts
const width = whenFeaturePropertyIn(
  "status",
  ["warning", "checking"],
  getFeatureNumber("activeWidth", 6),
  getFeatureNumber("width", 3)
);
```

## 属性映射

```ts
import { matchFeatureProperty } from "vue-maplibre-kit/business";

const color = matchFeatureProperty(
  "kind",
  {
    point: "#f97316",
    line: "#0f766e",
    area: "#79b8ff",
  },
  "#64748b"
);
```

映射结果也可以是表达式值：

```ts
const color = matchFeatureProperty(
  "kind",
  {
    point: getFeatureColor("pointColor", "#f97316"),
    line: getFeatureColor("lineColor", "#0f766e"),
    area: getFeatureColor("areaColor", "#79b8ff"),
  },
  getFeatureColor("color", "#64748b")
);
```

简单状态优先使用这些工具；复杂表达式可以继续使用 [原生表达式](./05-原生表达式.md)。
