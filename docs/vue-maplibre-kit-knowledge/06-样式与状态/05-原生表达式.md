# 原生表达式

适合谁读：表达式工具不够用，需要直接写 MapLibre 原生表达式的开发者。

先读哪篇：[02-表达式工具](./02-表达式工具.md)。

参考资料：[MapLibre 官方表达式参考](https://maplibre.org/maplibre-style-spec/expressions/)。

## 什么时候需要原生表达式

优先使用 `getFeatureColor()`、`getFeatureNumber()`、`createFeatureStateExpression()`、`matchFeatureProperty()` 这些业务工具。

当需要同时按多个字段、缩放级别、复杂条件或数学计算控制样式时，再写原生表达式。

## 基础结构

MapLibre 表达式是数组，第一个元素是操作名，后面是参数。

```ts
["get", "color"]
```

这表示读取当前要素的 `properties.color`。

## 取属性并提供默认值

```ts
["coalesce", ["get", "color"], "#79b8ff"]
```

`coalesce` 会返回第一个不是 `null` 的值。业务数据缺少字段或字段值为 `null` 时，建议为样式字段提供默认值。

如果业务数据可能出现空字符串，例如 `properties.color: ""`，需要显式判断后再回退：

```ts
[
  "case",
  ["==", ["get", "color"], ""],
  "#79b8ff",
  ["coalesce", ["get", "color"], "#79b8ff"]
]
```

## 类型转换

颜色字段推荐写成：

```ts
["to-color", ["coalesce", ["get", "color"], "#79b8ff"]]
```

数值字段推荐写成：

```ts
["to-number", ["coalesce", ["get", "width"], 3]]
```

字符串字段推荐写成：

```ts
["to-string", ["coalesce", ["get", "label"], ""]]
```

布尔字段推荐写成：

```ts
["to-boolean", ["coalesce", ["get", "disabled"], false]]
```

这些写法分别接近 `getFeatureColor()`、`getFeatureNumber()`、`getFeatureString()`、`getFeatureBoolean()`。业务 helper 还会额外处理空字符串，让空字符串也回退到默认值。

## 条件判断

```ts
[
  "case",
  ["==", ["get", "status"], "warning"],
  "#ef4444",
  "#22c55e"
]
```

这表示 `properties.status` 等于 `warning` 时使用红色，否则使用绿色。

## 多值映射

```ts
[
  "match",
  ["get", "kind"],
  "pipe",
  "#1677ff",
  "area",
  "#79b8ff",
  "#64748b"
]
```

这表示按 `properties.kind` 映射颜色，最后一个值是默认颜色。

## feature-state

```ts
[
  "case",
  ["boolean", ["feature-state", "selected"], false],
  "#2563eb",
  ["boolean", ["feature-state", "hover"], false],
  "#facc15",
  "#79b8ff"
]
```

`feature-state` 适合 hover、selected、临时高亮这类不写回 GeoJSON 的状态。

## 在样式工厂中使用

```ts
import { createLineLayerStyle } from "vue-maplibre-kit/business";

const style = createLineLayerStyle({
  paint: {
    "line-color": ["to-color", ["coalesce", ["get", "color"], "#1677ff"]],
    "line-width": ["to-number", ["coalesce", ["get", "width"], 3]],
  },
});
```

简单场景优先写成：

```ts
import {
  createSimpleLineStyle,
  getFeatureColor,
  getFeatureNumber,
} from "vue-maplibre-kit/business";

const style = createSimpleLineStyle({
  color: getFeatureColor("color", "#1677ff"),
  width: getFeatureNumber("width", 3),
});
```

## 常见注意点

- `get` 只负责取值，不负责保证值的类型。
- 样式字段缺少属性值或值为 `null` 时，优先用 `coalesce` 给默认值；如果还要把空字符串当空值，额外加一层 `case` 判断。
- 颜色、宽度、透明度这类字段建议使用对应的类型转换表达式。
- 表达式可以嵌套，内层表达式会先计算。
- 复杂表达式可直接写到 `create*LayerStyle({ paint })`，简单点线面优先用 `createSimple*Style()` 加表达式工具。
