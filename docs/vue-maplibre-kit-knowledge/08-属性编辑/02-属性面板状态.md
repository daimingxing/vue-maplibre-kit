# 属性面板状态

适合谁读：需要把要素属性渲染成可编辑面板的开发者。

先读哪篇：[01-propertyPolicy](./01-propertyPolicy.md)。

对应示例：[NGGI05](../../../examples/views/NG/GI/NGGI05.vue)。

## 解析当前编辑目标

```ts
import { computed, ref } from "vue";
import type { MapFeaturePropertyEditorTarget } from "vue-maplibre-kit/business";

const currentTarget = ref<MapFeaturePropertyEditorTarget | null>(null);

const editorState = computed(() => {
  return businessMap.editor.resolveEditorState(currentTarget.value);
});
```

## 面板数据

```ts
const panelItems = computed(() => editorState.value.panelState.items);
const rawProperties = computed(() => editorState.value.rawProperties);
```

`panelState.items` 中每个字段会带上 `editable`、`removable` 等状态，业务 UI 可以直接据此禁用输入框或隐藏删除按钮。

## 从点击生成目标

```ts
/**
 * 将点击上下文转换为属性编辑目标。
 * @param context 地图交互上下文
 */
function selectFeature(context: MapLayerInteractiveContext): void {
  const view = businessMap.feature.toBusinessContext(context);

  currentTarget.value = view.featureRef
    ? { type: "map", featureRef: view.featureRef }
    : null;
}
```

面板展示可以使用点击上下文里的属性，但保存前应通过 `featureRef` 回源读取最新要素。
