# 保存与删除属性

适合谁读：需要在属性面板中保存字段、删除字段的开发者。

先读哪篇：[02-属性面板状态](./02-属性面板状态.md)。

对应示例：[NGGI05](../../../examples/views/NG/GI/NGGI05.vue)。

## 保存单字段

```ts
/**
 * 保存当前编辑目标的名称。
 * @param name 最新名称
 */
function saveName(name: string): void {
  if (!currentTarget.value) {
    return;
  }

  const result = businessMap.editor.saveItem(currentTarget.value, {
    key: "name",
    value: name,
  });

  console.log(result.message);
}
```

`saveItem` 会根据目标类型和 `propertyPolicy` 判断能否写入。被保护或只读的字段不会被强行覆盖。

## 删除字段

```ts
/**
 * 删除当前编辑目标的 editable 字段。
 */
function removeEditable(): void {
  if (!currentTarget.value) {
    return;
  }

  const result = businessMap.editor.removeItem(currentTarget.value, "editable");
  console.log(result.message);
}
```

只有被规则判定为 `removable` 的字段才允许删除。示例中 `id`、`name`、`status` 会被保留，`editable` 允许删除。

## 直接使用 registry

不需要统一编辑器状态时，也可以直接写 source registry：

```ts
registry.saveProperties("asset-source", "asset-a", { name: "新名称" });
registry.removeProperties("asset-source", "asset-a", ["editable"]);
```

业务表单场景优先使用 `businessMap.editor`，批量后台写回可以使用 registry。
