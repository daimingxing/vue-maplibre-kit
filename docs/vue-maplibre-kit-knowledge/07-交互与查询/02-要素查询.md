# 要素查询

适合谁读：点击地图后，需要回到正式业务 source 读取最新要素的开发者。

先读哪篇：[01-点击和hover](./01-点击和hover.md)。

对应示例：[NGGI05](../../../examples/views/NG/GI/NGGI05.vue)。

## 推荐入口

```ts
const businessMap = useBusinessMap({
  mapRef: () => mapRef.value,
  sourceRegistry: registry,
});
```

## 从点击上下文转业务上下文

```ts
const businessContext = businessMap.feature.toBusinessContext(context);

if (businessContext.featureRef) {
  const latestFeature = businessMap.feature.resolveFeature(businessContext.featureRef);
}
```

`toBusinessContext` 会把原始交互上下文整理为业务友好的对象，包含 `featureRef`、`feature`、`properties`、`featureId`、`geometryType` 等字段。

## 标准来源引用

`MapSourceFeatureRef` 用来稳定表达“哪个 source 的哪个 feature”：

```ts
const featureRef = businessMap.sources.createFeatureRef(
  "asset-source",
  "asset-a",
  "asset-point-layer"
);
```

它适合在弹窗、表格行、属性面板和命令式动作之间传递目标。

## 查询当前选中

```ts
const selectedRef = businessMap.feature.getSelectedFeatureRef();
const selectedFeature = businessMap.feature.resolveSelectedFeature();
```

如果当前选中来自插件托管图层，查询门面会尽量返回可消费的快照；正式业务 source 仍优先回源读取最新数据。
