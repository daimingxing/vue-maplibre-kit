# 选中态

适合谁读：需要点击选中、清空选中或读取选中集的开发者。

先读哪篇：[02-要素查询](./02-要素查询.md)。

对应示例：[NGGI03](../../../examples/views/NG/GI/NGGI03.vue)、[NGGI10](../../../examples/views/NG/GI/NGGI10.vue)。

## 单选

普通图层交互在点击命中要素后，可以自动维护 `feature-state.selected`。

```ts
const interactive = {
  enabled: true,
  layers: {
    "asset-point-layer": {
      hitPriority: 30,
      enableFeatureStateSelected: true,
    },
  },
  onSelectionChange: (context) => {
    console.log(context.selectedCount);
  },
};
```

## 读取选中集

`onSelectionChange` 上下文提供完整选中集和快捷提取方法：

```ts
const selectedIds = context.getSelectedFeatureIds();
const selectedNames = context.getSelectedPropertyValues<string>("name");
```

## 清空选中

空白点击常用于清空业务面板。若需要主动清空地图选中，可通过公开实例或业务选择分组处理。

```ts
mapRef.value?.clearSelectedMapFeature();
```

## 多选

多选插件属于插件专题。本章只说明普通图层选中态；如果页面注册了多选插件，业务读取仍推荐走 `businessMap.plugins.multiSelect`。
