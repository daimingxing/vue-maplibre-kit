# 图层工厂

适合谁读：需要显式创建不同类型业务图层的开发者。

先读哪篇：[业务图层索引](./index.md)。

对应示例：[NGGI02](../../../examples/views/NG/GI/NGGI02.vue)。

## 四类业务图层

```ts
import {
  createCircleBusinessLayer,
  createFillBusinessLayer,
  createLineBusinessLayer,
  createSymbolBusinessLayer,
  createSimpleCircleStyle,
  createSimpleFillStyle,
  createSimpleLineStyle,
} from "vue-maplibre-kit/business";
```

| 工厂 | 对应 MapLibre 图层 |
| --- | --- |
| `createCircleBusinessLayer` | `circle` |
| `createLineBusinessLayer` | `line` |
| `createFillBusinessLayer` | `fill` |
| `createSymbolBusinessLayer` | `symbol` |

## 示例

```ts
const layers = [
  createFillBusinessLayer({
    layerId: "area-layer",
    geometryTypes: ["Polygon", "MultiPolygon"],
    style: createSimpleFillStyle({ color: "#79b8ff", opacity: 0.35 }),
  }),
  createLineBusinessLayer({
    layerId: "line-layer",
    geometryTypes: ["LineString", "MultiLineString"],
    style: createSimpleLineStyle({ color: "#0f766e", width: 4 }),
  }),
  createCircleBusinessLayer({
    layerId: "point-layer",
    geometryTypes: ["Point", "MultiPoint"],
    style: createSimpleCircleStyle({ color: "#f97316", radius: 7 }),
  }),
];
```

## 字段说明

- `layerId`：图层唯一 ID。
- `style`：图层 `layout` 和 `paint`。
- `geometryTypes`：限制几何类型。
- `where`：简单等值过滤。
- `filter`：原生 MapLibre 过滤表达式。
- `propertyPolicy`：当前图层的属性治理规则。
