# 图层过滤

适合谁读：需要让同一份 source 按几何类型或属性拆成多个图层的开发者。

先读哪篇：[01-图层工厂](./01-图层工厂.md)。

对应示例：[NGGI02](../../../examples/views/NG/GI/NGGI02.vue)。

## geometryTypes

`geometryTypes` 用来按 GeoJSON 几何类型过滤。

```ts
const lineLayer = createLineBusinessLayer({
  layerId: "pipe-line-layer",
  geometryTypes: ["LineString", "MultiLineString"],
});
```

多几何类型会折叠为 MapLibre 可识别的基础类型，例如 `MultiLineString` 归入线类型。

## where

`where` 适合简单等值条件。

```ts
const warningLayer = createCircleBusinessLayer({
  layerId: "warning-point-layer",
  geometryTypes: ["Point"],
  where: { status: "warning" },
});
```

## filter

复杂表达式使用 MapLibre 原生 `filter`。

```ts
const plannedLayer = createLineBusinessLayer({
  layerId: "planned-line-layer",
  geometryTypes: ["LineString"],
  filter: ["in", ["get", "status"], ["literal", ["planned", "checking"]]],
});
```

## 组合顺序

最终过滤会按 `geometryTypes -> where -> filter` 组合。三个条件同时存在时，需要全部命中才会渲染。
